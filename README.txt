// Resumen //
    Este juego es una imitacion de mario donde solo se muere al impactar con un enemig.
    Por lo que entiendo, state machine es como crear el constructor de un objeto dejando variables que luego podran usar los hijos.
    En el caso de Mario podria hacer un state machine para todos los enemigos menos browser ya que todos matan a mario al tocarlo y tambien se mueven de derecha a izquierda chocando con las paredes (menos las carnivoras y la nubecita)
